This is a small example of how to make an acoustic drum sample trigger with Vanilla JavaScript, CSS3 and HTML5. In the desktop version you can shoot the samples with the keyboard. Each element has a key next to it that triggers the sounds. In the mobile version the pads are tactile. Felt free to use either version. Enjoy.
